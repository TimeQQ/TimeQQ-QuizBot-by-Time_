Quiz Bot — Telegram викторина с бесконечными вопросами
Telegram бот для проведения викторин. Вопросы загружаются из открытой базы Open Trivia Database (opentdb.com) при каждом запуске игры. Бот ведёт счёт, сохраняет результаты в SQLite и показывает таблицу лидеров.

Возможности
Бесконечное количество вопросов — каждый раз новые, из более чем 4000 доступных.
Разнообразные категории: программирование, история, наука, кино, игры и другие.
Варианты ответов с инлайн-кнопками.
Подсчёт правильных ответов и сохранение общей статистики пользователя.
Таблица лидеров с суммой баллов, количеством игр и средним результатом.
Простота запуска и использования.

Технологии
Python 3.9+
Aiogram 3.x — асинхронный фреймворк для Telegram Bot API
Aiohttp — для запросов к Open Trivia Database
SQLite — хранение статистики пользователей
FSM (Finite State Machine) — управление состоянием игры

Установка и запуск
Клонируйте репозиторий
git clone https://github.com/yourusername/quiz-bot.git
cd quiz-bot

Установите зависимости
pip install aiogram aiohttp

Получите токен бота
Напишите @BotFather в Telegram
Создайте нового бота и скопируйте токен

Настройте токен в коде
Откройте файл main.py и замените YOUR_BOT_TOKEN на полученный токен.

Запустите бота
python main.py
Команды бота
/start — приветственное сообщение и краткая инструкция.
/quiz — начать новую викторину (10 вопросов).
/leaderboard — показать таблицу лидеров (топ-10 игроков).

Пример работы
Пользователь отправляет /quiz.
Бот загружает 10 случайных вопросов из Open Trivia Database.
На каждый вопрос предлагается 4 варианта ответа в виде инлайн-кнопок.
После выбора бот сообщает, правильный ответ или нет, и показывает следующий вопрос.
По окончании игры бот выводит количество правильных ответов и обновляет статистику.
Команда /leaderboard показывает общую таблицу лидеров.
